about=This plugin provides a QGIS Processing interface for the ML Vegetation Filter tool (Wernette 2024, doi:10.5281/zenodo.10966854). It classifies LAS/LAZ point cloud points as vegetation or bare-Earth using RGB colour-derived vegetation indices and an MLP neural network trained via TensorFlow. Three workflows are provided: (1) Train a new model, (2) Reclassify a point cloud with an existing model, and (3) Train and reclassify in a single step.
